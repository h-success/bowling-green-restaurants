Country Ham Days Festival in Lebanon, KY

New Maps Plus

Authored by Heather Cowherd

The map above represents the location of the Country Ham Days Festival which typically occurs each year in the downtown area of Lebanon, KY. Points of interests, buildings, and street names can be seen when the map is zoomed in. Also, all of the surrounding counties can be veiewed when the map is zoomed out.

Lebanon, KY is described as being the Heart of Kentucky. The Country Ham Days Festival is one of the Top Ten Events that occur in Kentucky.

According to Kentucky Living, the festival involves a country ham breakfast, country concerts, various crafts/food vendors, 5K races, and the Pigasus Parade. The festival began in 1969 and typically has over 40,000 attendees. Since 2018, over 4,000 pounds of country ham have been served at the festival's annual country ham breakfast.

Data Sources

Information about Country Ham Days Festival was extracted from the Lebanon: Heart of Kentucky's and Kentucky Living's websites.

A shape file containing information about the US counties was extracted from the US Census website. QGIS 3.14.16 was used to create a Centroids layer and to convert the file to a GeoJSON so that it can be used as a dataset in Mapbox.

Tools used in this project

  • Data is shown on Mapbox web map with geolocation services enabled. If the map got popular it would require a fee to use.
  • Mapbox's tilesets were used to create map layers.
  • The points of interests, place, and road labels use BellTopo Sans designed by Sarah Bell, a cartographer inspired by historic USGS maps.
